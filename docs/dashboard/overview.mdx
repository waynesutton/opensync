## Session List

The main sessions list shows:

- **Session title** - Auto-generated from the first prompt
- **Source** - Which tool the session came from (OpenCode, Claude, Codex, Cursor)
- **Model** - The AI model used
- **Tokens** - Total tokens (prompt + completion)
- **Cost** - Estimated cost based on model pricing
- **Time** - When the session was created

## Filters

Filter sessions by:

- **Source** - Filter by tool (OpenCode, Claude Code, Codex, Cursor)
- **Date range** - Filter by time period
- **Search** - Full-text search across all session content

## Session Details

Click any session to view:

- Complete conversation history
- Message-by-message token counts
- Copy as markdown
- Export options
