## Install Plugins Locally

Install one or more plugins globally. They run on your machine and sync to the hosted dashboard.

### OpenCode

```bash
npm install -g opencode-sync-plugin
```

[View on npm](https://www.npmjs.com/package/opencode-sync-plugin)

### Claude Code

```bash
npm install -g claude-code-sync
```

[View on npm](https://www.npmjs.com/package/claude-code-sync)

### Codex CLI

```bash
npm install -g codex-sync
```

[View on npm](https://www.npmjs.com/package/codex-sync)

### Cursor

```bash
npm install -g cursor-sync-plugin
```

[View on npm](https://www.npmjs.com/package/cursor-opensync-plugin)

## Login and Sync

<Steps>
  <Step title="Sign in">
    Sign in at [opensync.dev](https://www.opensync.dev)
  </Step>
  <Step title="Generate API Key">
    Go to **Settings** and click **Generate API Key**
  </Step>
  <Step title="Run login command">
    Run the login command for your plugin: ```bash opencode-sync login # or
    claude-code-sync login # or codex-sync login # or cursor-sync login ```
  </Step>
  <Step title="Enter credentials">
    Enter the **Convex URL** and **API key** when prompted
  </Step>
  <Step title="Start coding">Start coding. Sessions sync automatically.</Step>
</Steps>
